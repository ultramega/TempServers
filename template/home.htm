Welcome to TempServers!

Our mission is to provide low cost game server rentals for hourly rates instead of the traditional monthly rates of other game server providers. If you need a temporary game server, this is the place to get it!

About TempServers

We have built a completely automated system from the ground up. It is designed to be simple and easy to use, while giving you much control over your server.

Reserving a server is easy: just log in to your account, choose a time, choose a game, and check out.

You have access to the server configuration to change things such as the server name, password, and any other settings. You will also be able to reboot or change the game running on your server on the fly!

TempServers is hosted and managed by UltraMega Tech.
